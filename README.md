# SkillSwap
*Conectando conocimiento estudiantil.*

## Descripción del Proyecto
SkillSwap es una plataforma diseñada para facilitar el intercambio de tutorías entre estudiantes universitarios. Permite que alumnos de semestres superiores (tutores) ayuden a quienes inician su carrera (estudiantes), optimizando el aprendizaje y reduciendo la deserción.

## Equipo y Roles
* *Integrante 1:* Product Owner | Diseñador UI/UX
* *Integrante 2:* Líder Técnico | Arquitecto de Software
* *Integrante 3:* Admin. Base de Datos | Analista de QA

## Tecnologías Propuestas
* *Frontend:* React / Next.js
* *Backend:* Node.js / Express
* *Base de Datos:* PostgreSQL / MongoDB

La base de datos del proyecto se encuentra en: database/skillswap_schema.sql
